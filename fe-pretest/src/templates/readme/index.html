UI / UX

  • 我們提供 React 以及 Vue 的前端框架讓你使用，使用方式請參照 參考資料
  • 設計儀表板畫面，尺寸最大為 1024 x 768
  • 請考慮到使用者的閱讀性，根據設定檔的需求顯示使用者需要得到的資訊

資料更新

  • 我們提供 HTTP 請求插件 Axios 讓你使用，使用方式請參照 參考資料
  • 請使用設定檔的 pcm_id、devid 放進 Request Payload，並以POST的方式分批至 API 換取資料
  • 每 5 秒擷取新資料

  • API: LOADING
